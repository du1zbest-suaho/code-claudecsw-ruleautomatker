# 코드 체계 참조 문서

## 보험기간 코드 체계

| 만기 유형 | ISRN_TERM_DVSN_CODE | ISRN_TERM | ISRN_TERM_INQY_CODE | 예시 |
|----------|---------------------|-----------|---------------------|------|
| 년만기 | N | 숫자 | N{숫자} | 20년만기 → N, 20, N20 |
| 세만기(주피보험자) | X | 숫자 | X{숫자} | 90세만기 → X, 90, X90 |
| 종신 | A | 999 | A999 | 종신 → A, 999, A999 |
| 세만기(자녀) | Z | 숫자 | Z{숫자} | 30세만기(자녀) → Z, 30, Z30 |
| 세만기(피부양자) | V | 숫자 | V{숫자} | 30세만기(피부양자) → V, 30, V30 |
| 세만기(계약자) | W | 숫자 | W{숫자} | 65세만기(계약자) → W, 65, W65 |

## 파일명 패턴

### 사업방법서
`한화생명_{상품명}_{YYYYMMDD}.zip`
- YYYYMMDD: 효력발생일 → VALID_START_DATE

### 출력 파일
| 파일 | 패턴 |
|------|------|
| 추출 결과 | `{upper_obj}_{table_type}_{run_id}.json` |
| 코드 변환 결과 | `{upper_obj}_{table_type}_{run_id}_coded.json` |
| 검증 리포트 | `{upper_obj}_{table_type}_{run_id}_report.json` |
| 업로드 양식 | `{table_type}_{upper_obj}_{run_id}.xlsx` |
